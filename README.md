# Ben Bot

Discord bot of Talking Ben that i made because i was bored.


## Run Locally

Clone the project

```bash
  git clone https://github.com/puxt/Ben-Bot
```

Go to the project directory

```bash
  cd Ben-Bot
```

Install dependencies

```bash
  pip install -U discord.py
```

Start the Bot

```bash
  python bot.py
```
